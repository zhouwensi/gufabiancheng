# 古法编程 | Artisanal Game Coding

🌐 **https://artisanalcoding.com**

---

## 这是什么

在 AI 生成游戏的时代，记录和传承那些即将失传的**纯手工游戏编程**技艺。

手写引擎、手绘像素、手调手感、手搓物理——这些曾经每个游戏开发者的日常，正在成为"非物质文化遗产"。

## 本站技术栈

纯古法打造：

- 纯手写 HTML / CSS / JavaScript
- 无框架、无构建工具、无 npm
- 像素风设计
- 部署于 GitHub Pages
